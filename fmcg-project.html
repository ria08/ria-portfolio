FMCG Sales Automation Pipeline
Project Walkthrough
1 / 9

A regional FMCG distributor was losing 80–115 minutes every morning to manual sales reporting. Consolidating CSVs by hand. Cleaning dirty data. Rebuilding charts. Exporting reports. Repeated every single working day.

This pipeline eliminates all of it. Python generates fresh data at 6:30 AM via Task Scheduler. Power Query ingests and cleans every new file automatically. Three VBA macros handle the rest. Power BI surfaces everything in one executive dashboard. Zero manual work after setup — saving 30–40 hours a month.

The data shown here is updated till March 10, and it is growing every day.
