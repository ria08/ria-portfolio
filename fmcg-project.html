FMCG Sales Automation Pipeline
Python Power Query Excel VBA Power BI Task Scheduler DAX
What this project does
Fully automated pipeline — Python drops a fresh CSV at 6:30 AM via Task Scheduler. No manual trigger ever needed.
Self-cleaning data — Power Query removes duplicates, fixes nulls, strips bad rows and calculates revenue on every new file automatically.
Live EDA layer — Six structured blocks in Excel update the moment new data loads. No formula editing required, ever.
Audit trail built in — VBA logs every refresh with timestamp, row count, and data quality metrics.
Executive dashboard — One-page Power BI with 5 KPI cards, daily trend, breakdowns, and a region slicer.
