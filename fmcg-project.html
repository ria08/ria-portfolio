FMCG Sales
Automation Pipeline

A regional FMCG distributor was losing 80–115 minutes every morning to manual sales reporting. Consolidating CSVs by hand. Cleaning dirty data. Rebuilding charts. Exporting reports. Repeated every single day.

This pipeline eliminates all of it. Python generates the data automatically at 6:30 AM. Power Query ingests and cleans every new file. VBA handles the rest. Power BI surfaces everything in one dashboard. Zero manual work after setup — saving 30–40 hours a month.

A new file.
Every morning.
Automatically.
Task Scheduler wakes up at 6:30 AM and fires the Python script. A fresh dated CSV appears in the folder before the analyst's coffee is ready.

These file timestamps are the most honest proof the automation works. No human triggered these.
04 — Data Cleaning
15 cleaning steps.
Written once.
Run forever.
Power Query connects to the folder — not a file. Every new CSV that drops gets processed through the same steps automatically.
  • Removes duplicate order IDs
  • Strips negative quantity rows
  • Fixes blank customer names and regions
  • Calculates revenue and net revenue
05 — Live EDA Layer
₹1.66 Crore.
