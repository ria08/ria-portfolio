FMCG Sales Automation Pipeline

A fully automated daily sales reporting pipeline for a regional FMCG distributor — from raw CSV to executive dashboard with zero manual intervention.

Python Power Query Excel VBA Power BI Windows Task Scheduler DAX
What this project does
Fully automated pipeline — Python generates a fresh daily CSV at 6:30 AM via Windows Task Scheduler. No manual trigger needed.
Self-cleaning data — Power Query ingests every new file, removes duplicates, fixes nulls, strips bad rows, and calculates revenue in one reusable pipeline.
Live EDA layer — Six structured summary blocks in Excel auto-update the moment new data loads. No formula editing required, ever.
Audit trail built in — A VBA macro logs every refresh with timestamp, row count, and data quality metrics. Every run is on record.
Executive dashboard — One-page Power BI report with 5 KPI cards, daily trend, category and salesperson breakdowns, and a region slicer.

Project walkthrough
The pipeline is live — new CSV every morning
Each dated file was dropped automatically by Task Scheduler at 6:30 AM. This folder is the only manual-free proof you need that the automation works.
15 cleaning steps — written once, run forever
Power Query removes duplicates, fixes nulls, strips negative quantities, calculates revenue and net revenue. Every new CSV gets the same treatment automatically.
Live EDA layer — 6 blocks, zero manual updates
8,353 orders, ₹1.66 Cr revenue, 5.15% return rate. All structured references to the sales_data table. When new data loads, every number updates instantly.
Audit trail — every run logged automatically
Row counts climb ~500 per day. Unknown customers and unassigned regions stay at zero — confirming Power Query catches every quality issue before it reaches the analysis layer.
Executive dashboard — everything on one screen
5 KPI cards, daily revenue trend, category and salesperson breakdowns, channel donut, and a region slicer. One-click refresh pulls latest data from Excel.
